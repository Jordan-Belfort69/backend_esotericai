# Картинки для бэкенда

## Совет дня (рассылка)

- **`advice/post.png`** — картинка поста «Совет Арканы на завтра» (три карты + луна, как на скриншоте).  
  Без этого файла скрипт рассылки выведет предупреждение и не отправит фото.

- **`cards/`** — изображения карт Таро для ответа на нажатие кнопок 1/2/3.  
  Имена файлов задаются в `app/data/daily_advice_cards.py` (поле `image` у каждой карты), например:
  - `page_pentacles.png`
  - `sun.png`
  - `star.png`
  - `magician.png`
  - `high_priestess.png`
  и т.д. Добавьте свои карты в список и положите соответствующие PNG в эту папку.
